# Customer Insights & Recommendation Dashboard
**ReadyNest Corp. | Data Analytics Internship — Week 2 | June 2025**

---

## Project Overview
Analyzed customer and sales data to uncover actionable business insights. Built an end-to-end data analytics pipeline — from raw data generation and cleaning to an interactive Power BI dashboard and a professional business insights report.

---

## Project Name
Customer Insights & Recommendation Dashboard — ReadyNest Corp.

---

## Architecture
```
Raw Data → Cleaning & Preprocessing → EDA → Segmentation → Dashboard → Report
```

---

## Modules Built

| Module | Description |
|---|---|
| Data Generation | 500 customers, 1,800 orders, 50 products, 5,000+ sales records |
| Data Cleaning | Null handling, duplicate removal, outlier correction |
| EDA Notebook | 9 charts across customer, sales, product & behaviour analysis |
| Customer Segmentation | Spend-based High / Medium / Low Value segmentation |
| Power BI Dashboard | 3-page interactive dashboard with KPIs, slicers & charts |
| Business Insights Report | 8-section report with 7 actionable recommendations |

---

## Tech Stack
- **Language:** Python 3.14
- **Libraries:** pandas, NumPy, matplotlib, seaborn
- **Notebook:** Jupyter
- **Dashboard:** Power BI Desktop
- **Report:** Microsoft Word
